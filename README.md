# odin-recipes
## DESCRIPTION:

### Current Project
This will be a website with links to recipe pages. The recipe pages will display an image of the dish, a description of the dish, a list of ingredients, and directions to make the dish. There will also be a link back to the home page.

### Skills to be Demonstrated
- Create HTML boilerplate
- Create multiple connected webpages
- Organize website directories and files
- Create README.md file
- Use of HTML tags such as:
    - Headings: `<h1>`
    - Links: `<a>`
    - Images: `<img>`
    - Lists: `<ol>` and `<ul>` along with `<li>`
- Git Practice
    - Create repository
    - Clone repository
    - Commits/Commit Comments

## PROJECT STRUCTURE

### Main Project Folder
- `index.html`: The home page of the website that lists recipe links.
- `README.md`: Project documentation, including information about the current plan for the project, the skills to be demonstrated during the process of building the project, and the project structure.

### recipes
- This direction includes html pages. Each page is a recipe with a picture, directions, and a link back to the homepage (`index.html`).
